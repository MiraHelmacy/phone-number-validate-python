# phone-number-validate-python

## Usage
```
usage: phone-number-validate [-h] [--invalidonly | --validonly] [--short]
                             [--cfg CFG] [--version]
                             [phoneNumbers ...]

positional arguments:
  phoneNumbers   phone numbers to validate

options:
  -h, --help     show this help message and exit
  --invalidonly  only show invalid results
  --validonly    only show valid results
  --short        Short Output only.
  --cfg CFG      config file with phone numbers
  --version      print version information.
```
